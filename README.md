# 📌 Overview

**Student Data Organizer** is a Python-based command-line application that helps users manage student information efficiently.

This project allows you to:

✅ Add student records
✅ Display all students
✅ Update student details
✅ Delete student records
✅ View unique subjects offered
✅ Store multiple subjects for each student

It is designed for beginners who want to practice:

* Python Fundamentals
* List & Dictionary Operations
* Functions
* CRUD Operations
* User Input Handling
* Console Application Design

---

# 🚀 Features

## 📝 Add Student

Add complete student details including:

* Student Name
* Age
* Grade
* Subjects
* Student ID
* Date of Birth

## 📋 Display All Students

Shows all student records in a formatted structure.

Remove student records permanently.
